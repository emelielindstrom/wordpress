Show your users which Wordpress themes are installed with a simple shortcode. On Multisite installations only 'activated' themes will be displayed.

== Description ==

This plugin provides a shortcode to display the screenshots of all installed themes in your blog. On Multisite installations only 'activated' themes will be displayed. This plugin can be used to offer a theme preview like http://theme.wordpress.com/

== Installation ==

1. Upload the sxss Theme showcase plugin to the /wp-content/plugins/ folder.
2. Activate it.
3. Use the shortcode [themes] (options are displayed in the FAQ section)

== Frequently Asked Questions ==

= Can i controll how many theme screenshots will be displayed? =

Yes. I.e. you can use '[themes quantity="5"]' to display 5 screenshots.

= Can i change the size of the screenshots? =

Yes. I.e. you can use '[themes width="300"]' to display screenshots with a width of 300px. The height will be calculated in the proportion 4:3.
