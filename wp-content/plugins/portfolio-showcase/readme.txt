Easy to Use project/portfolio showcase management plugin for WordPress

== Description ==

Portfolio Showcase is a Easy to Use project/portfolio showcase management plugin for WordPress. Add portfolio item in different categories and show then just using a simple short-code.

= Features =
* Portfolio Category Support
* Category Filter Support using Mixitup
* Modal Preview for Portfolio Details
* Use simple short-code [wpeden_portfolio] to embed portfolio
* Translation Ready 
 
== Installation ==


1. Upload `portfolio-manager` to the `/wp-content/plugins/`  directory
2. Activate the plugin through the 'Plugins' menu in WordPress
